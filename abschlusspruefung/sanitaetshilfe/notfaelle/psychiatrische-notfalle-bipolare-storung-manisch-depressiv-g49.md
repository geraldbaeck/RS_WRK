# Psychiatrische Notfälle - Bipolare Störung (manisch-depressiv) (G49)

## Definition
Manische und depressive Phasen wechseln sich ab.

## Symptome
Depression:
+ gedämpfte Stimmung
+ Antriebslosigkeit
+ Ausdruck der Hoffnungslosigkeit
+ Konzentrationsstörungen
+ Suizidgedanken

Manie:
+ Gehobene Stimmung
+ Rededrang
+ Impulsivität
+ Antriebssteigerung
